Dispositivo para Surdocegos

Desenvolvi, em 2018, durante meu curso técnico de Informática, um dispositivo computacional microcontrolado para comunicação independente de pessoas surdocegas. O projeto foi realizado no âmbito do INIC (Iniciação Científica da Univap) e contou com a orientação do meu professor. Participei ativamente de todas as etapas do desenvolvimento, adquirindo conhecimento prático e interdisciplinar.

Ferramentas utilizadas:

  • Arduino
  • Tinkercad
  • MIT App Inventor

Sistema de Ocorrências

Desenvolvido em 2021 como Trabalho de Conclusão de Curso (TCC) e estágio no Colégios Univap. A aplicação tem como objetivo central substituir o livro de ocorrências físico. O projeto é dividido em 2 módulos:

Módulo do Professor - Aplicação web para registrar ocorrências e advertências diretamente pelo professor.

Ferramentas utilizadas:

  • ReactJs
  • Firebase

Sistema de Ocorrências

Módulo do Orientador - Aplicação web para administrar dados das ocorrências. Permite aos orientadores acessar o histórico completo de ocorrências de cada aluno, auxiliando nos processos administrativos e disciplinares.

Juntos, os módulos oferecem uma solução integrada para o monitoramento disciplinar na escola, promovendo uma comunicação mais clara entre os professores e orientadores.

Ferramentas utilizadas:

  • ReactJs
  • Firebase
